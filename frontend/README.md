# Pulse — Frontend

React 18 + Vite frontend for the Pulse employee growth platform.

See the [project README](../README.md) for full setup instructions, seed credentials, and architecture overview.

## Development

```bash
npm install
npm run dev
```

Runs on `http://localhost:5173`. Requires the backend running on port `8000`.

## Stack

- **React 18** + Vite
- **React Router v6** for client-side routing
- **Vanilla CSS** with design tokens (no Tailwind / UI frameworks)
- **Recharts** for growth trend charts
- **Lucide React** for icons
